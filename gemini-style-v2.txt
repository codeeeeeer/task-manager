Vue 3 + Element Plus 2.3 UI 实现规范本文档旨在指导如何在 Vue 3 项目中，基于 Element Plus 实现“明亮简约”与“高科技赛博”两套风格的无缝切换。1. 核心策略：CSS 变量主题化我们将使用 CSS 变量来控制颜色系统。Default (Root): 对应明亮风格。.theme-tech: 对应高科技风格，通过给 html 或 body 添加此类名来激活。1.1 主题变量定义 (src/styles/theme.scss)建议在项目入口 (main.js 或 App.vue) 引入此样式文件。:root {

2. 组件深度定制 (src/styles/element-overrides.scss)为了实现“高科技”风格特有的发光、切角和透明效果，我们需要对 Element Plus 组件进行 targeted overrides。/* 全局通用修正 
  这些样式会自动应用，并根据 CSS 变量变化
*/
body {
  font-family: var(--app-font-family);
  transition: background-color 0.3s, color 0.3s;
}

/* ====== 针对 .theme-tech 模式的特殊处理 ====== */
.theme-tech {

  /* 1. 卡片 (El-Card) - 磨砂玻璃效果 */
  .el-card {
    background: var(--el-bg-color-overlay);
    backdrop-filter: blur(10px);
    border: 1px solid var(--el-border-color);
    box-shadow: var(--app-shadow);
    color: var(--el-text-color-primary);
    
    /* 标题发光 */
    .el-card__header {
      border-bottom: 1px solid var(--el-border-color);
      text-transform: uppercase;
      letter-spacing: 1px;
      text-shadow: 0 0 5px rgba(6, 182, 212, 0.5);
    }
  }

  /* 2. 表格 (El-Table) - 透明化与数据感 */
  .el-table {
    --el-table-bg-color: transparent;
    --el-table-tr-bg-color: transparent;
    --el-table-header-bg-color: rgba(6, 182, 212, 0.05);
    --el-table-border: 1px solid var(--el-border-color);
    --el-table-text-color: var(--el-text-color-regular);
    --el-table-header-text-color: var(--el-color-primary);

    th.el-table__cell {
      font-family: 'JetBrains Mono', monospace;
      font-weight: bold;
      letter-spacing: 1px;
    }
    
    /* 悬停行高亮 */
    .el-table__body tr:hover > td {
      background-color: rgba(6, 182, 212, 0.1) !important;
      box-shadow: inset 0 0 10px rgba(6, 182, 212, 0.2);
    }
  }

  /* 3. 按钮 (El-Button) - 辉光与直角 */
  .el-button {
    border-radius: var(--app-button-shape);
    text-transform: uppercase;
    letter-spacing: 1px;
    transition: all 0.3s ease;
  }

  .el-button--primary {
    box-shadow: 0 0 8px rgba(6, 182, 212, 0.4);
    border: 1px solid var(--el-color-primary);
    background: rgba(6, 182, 212, 0.2); /* 半透明背景 */
    
    &:hover {
      background: rgba(6, 182, 212, 0.4);
      box-shadow: 0 0 15px rgba(6, 182, 212, 0.6);
      text-shadow: 0 0 2px #fff;
    }
  }

  /* 4. 输入框 (El-Input) - 科技线框 */
  .el-input__wrapper {
    background-color: rgba(15, 23, 42, 0.6);
    box-shadow: 0 0 0 1px var(--el-border-color) inset;
    border-radius: 0;
    
    &.is-focus {
      box-shadow: 0 0 0 1px var(--el-color-primary) inset, 0 0 10px rgba(6, 182, 212, 0.2);
    }
  }

  /* 5. Tag/Badge - 荧光效果 */
  .el-tag {
    border-radius: 0;
    border-width: 1px;
    background-color: transparent;
    
    &.el-tag--success {

      border-color: var(--el-color-warning);
      color: var(--el-color-warning);

    }
  }
}

4. 关键实施建议Tailwind CSS 配合: 上述示例结合了 Tailwind CSS 的 Utility classes (如 flex, h-screen, bg-[var(--el-bg-color)])。推荐使用 Tailwind 来处理布局和间距，而让 Element Plus 及其 CSS 变量专注于组件样式的呈现。字体选择:明亮风格：推荐使用系统默认无衬线字体。高科技风格：强烈建议引入一款等宽字体（如 'JetBrains Mono' 或 'Roboto Mono'）用于数字显示和表格内容，这会瞬间提升“仪表盘”的感觉。动画过渡: 在 CSS 中为 background-color, border-color, color 添加 transition: ... 0.3s 是切换风格时不刺眼的关键。
