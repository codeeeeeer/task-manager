# 任务分发工具 - Chrome插件

## 说明

这是任务分发工具的Chrome浏览器插件，用于实时接收任务通知。

## 安装方法

1. 打开Chrome浏览器
2. 进入 `chrome://extensions/`
3. 启用"开发者模式"（右上角开关）
4. 点击"加载已解压的扩展程序"
5. 选择当前文件夹（chrome-extension）

## 使用说明

1. 安装后点击浏览器右上角的插件图标
2. 点击"设置"按钮
3. 填写服务器地址和用户UM编号
4. 保存设置

## 注意事项

- 需要先部署后端服务器
- 服务器地址格式: `http://IP:端口`，例如 `http://192.168.1.100:5000`
- 用户编号需要与后台系统中的UM编号一致

## 图标资源

请在 `assets` 文件夹中放置以下图标文件：
- icon16.png (16x16像素)
- icon48.png (48x48像素)
- icon128.png (128x128像素)
