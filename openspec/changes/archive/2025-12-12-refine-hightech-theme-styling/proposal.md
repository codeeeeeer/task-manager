# Proposal: Refine Theme Styling

## Change ID
`refine-hightech-theme-styling`

## Summary
根据 gemini-style.md 中定义的设计规范,优化现有的两种主题样式:高科技赛博风和明亮简约风,使其更符合设计规范中的色彩系统、元素定义和视觉效果要求。

## Motivation
当前系统已实现主题切换功能,但现有样式与 gemini-style.md 中定义的设计规范存在差异:

### 高科技主题问题:
1. **色彩系统不一致**: 当前使用 `#00ffff` (青色) 作为主色调,但设计规范建议使用 `#06B6D4` (霓虹蓝) 或 `#8B5CF6` (电光紫)
2. **背景色偏差**: 当前全局背景为 `#0a0e27`,设计规范要求 `#0B1120` (深蓝黑)
3. **元素样式细节**: 卡片圆角、边框样式、辉光效果等需要按照设计规范进行调整
4. **缺少特定效果**: 设计规范要求的磨砂玻璃效果 (backdrop blur)、切角设计等未完全实现

### 默认主题问题:
1. **色彩系统需优化**: 需要按照 gemini-style.md 中"明亮简约风"的色彩规范调整
2. **背景色调整**: 全局背景应为 `#F3F4F6` (极浅灰),卡片背景为 `#FFFFFF` (纯白)
3. **主色调统一**: 品牌色应为 `#3B82F6` (标准蓝),悬停色为 `#2563EB` (深蓝)
4. **元素样式优化**: 卡片阴影、按钮圆角、表格样式等需要符合简约风格规范

## Goals
- 将高科技主题的色彩系统和视觉效果调整为符合 gemini-style.md 中"高科技赛博风"规范
- 将默认主题的色彩系统和视觉效果调整为符合 gemini-style.md 中"明亮简约风"规范
- 优化两种主题下卡片、按钮、表格等核心元素的视觉样式
- 确保所有 Element Plus 组件在两种主题下的样式一致性

## Non-Goals
- 不改变主题切换的交互逻辑
- 不涉及功能性变更,仅限视觉样式调整
- 不添加新的主题类型

## Affected Components
- `frontend/src/assets/styles/themes.scss` - 主题样式定义文件
- `frontend/src/views/Home.vue` - 首页驾驶舱样式
- 其他使用主题变量的 Vue 组件 (如需要)

## Dependencies
- 依赖现有的主题切换机制 (theme store)
- 依赖 Element Plus 组件库的 CSS 变量覆盖能力

## Risks and Mitigations
- **风险**: 样式调整可能影响用户体验的连续性
  - **缓解**: 保持主题切换功能不变,用户可随时切换回明亮主题
- **风险**: 某些 Element Plus 组件样式覆盖可能不完全
  - **缓解**: 通过测试覆盖所有使用的组件,确保样式一致性

## Open Questions
- 高科技主题主色调选择: 使用 `#06B6D4` (霓虹蓝) 还是 `#8B5CF6` (电光紫)?
  - 建议: 保持当前的青色系 `#06B6D4`,因为已有用户习惯
- 是否需要为首页驾驶舱添加特殊的磨砂玻璃效果?
  - 建议: 在卡片上添加 backdrop-filter 实现磨砂效果
- 默认主题是否需要保留斑马纹表格样式?
  - 建议: 可选实现,偶数行使用极浅灰 `#F9FAFB`
